Pour ce projet j'ai utlilisé la librairie React JS et le CRA. Ainsi que le préprocesseur SASS et la librairie Bootstrap.
Pour initialisé le projet faire un NPM,YARN,PNMP install selon votre gestionnaire de package. Ensuite lançer la commande "yarn start" pour démarrer le projet.
Ouvrir avec n'importe quelle navigateur sauf chrome !
